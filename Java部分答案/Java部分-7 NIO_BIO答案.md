- 2.IO流按照方向和数据类型划分能划分为哪些数据流？

> 按照方向划分：输入流【从文件输入到程序】 & 输出流【程序输出到文件】  
> 按照数据类型划分： 字节流【以字节为单位】 & 字符流【以字符为单位】
